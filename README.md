# Описание Playbook
```
Playbook сосотоит из трех play. 
Первый выполняет: скачивание пакета Elasticsearch на удаленный хост (el_instance), установку и производит настройку (копирует подготовленные конфигурационный файл elasticsearch.yml).
Второй выполняет скачивание пакета Kibana на удаленный хост (k_instance), установку и производит настройку (копирует подготовленные конфигурационный файл kibana.yml).
Третий выполняет: скачивание пакета Filebeat на удаленный хост (aplication_instance), установку, производит (копирует подготовленные конфигурационный файл kibana.yml), запускает модуль system для сбора параметров и загружает Kibana dashboard.
Playbook включает переменные  описанные в файлах каталогов group_vars и  inventory
```
